# 📊 Sales Performance Analysis Dashboard | Power BI

## 📌 Project Overview

The **Sales Performance Analysis Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI** to transform raw sales data into meaningful business insights. The dashboard helps analyze sales performance, monitor key business metrics, identify top-performing products, and support data-driven decision-making through dynamic visualizations.

---

## 🎯 Business Objectives

- Analyze overall sales performance.
- Track revenue and sales trends.
- Identify top-performing products and categories.
- Evaluate regional sales performance.
- Monitor key business KPIs.
- Enable interactive business reporting.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
